WPWS design system · mood boards · July 2026

Thirteen palettes, six roles, four voices

Each palette now carries a vibe — a plain-language brand feel a client can recognize themselves in without any design vocabulary. Twelve palettes are mapped across two axes — warm to cool, soft to bold — three to a quadrant. Every palette runs on the same six-role anatomy (Base, Paper, Ink, Lead, Spice, Wash), so any layout built on the tokens accepts any palette. Each board is rendered in its own colors and set in its quadrant's default type pairing, but color and type are two independent choices — pairings can be swapped freely (see Type pairings at the end). Every text pairing shown is WCAG-verified.

The spectrum

Two questions place a client on the map: should it feel more energizing or more calming? Warmer or cooler? Their answers point at a quadrant, and they choose between its three palettes — never from a lineup of twelve. Each quadrant now offers a distinct personality within its temperature: no two palettes compete for the same client.

Color and type are two independent dials. Each palette is shown below in its quadrant's default pairing, but any of the four proposed Google Fonts pairings works with any of the thirteen palettes. Pick the type that matches the brand's voice, not the color it happens to sit next to on the map. The four pairings meet the sourcing criteria (Google Fonts, true italics, tall x-height at 16px) and are proposals pending Whitney's confirmation: warm-soft Fraunces + Karla, warm-bold DM Serif Display + Plus Jakarta Sans, cool-soft Lora + Work Sans, cool-bold Instrument Serif + Archivo. See Type pairings at the end of the doc for the specimens.

Find your vibe

No brand vocabulary needed — pick the phrase that sounds like how you want clients to see you. Each one maps to exactly one palette.

Working rules

These apply to every palette. The role names stay identical across all twelve, so any layout built on the tokens accepts any palette.

Never one hue family

Lead and Spice sit on opposite sides of the wheel at roughly equal saturation. The Spice is the color that "shouldn't be there" — it makes the palette feel authored instead of safe.

No mud zone

Nothing lives at medium saturation and medium lightness. Every color is very light (Base, Paper, Wash), very dark (Ink), or clearly saturated (Lead, Spice).

Clean pastels

Calm palettes soften their accents, but the pastels stay clean — periwinkle, not dusty lilac. Graying down is not calming, it is dusty.

Base carries the temperature

Every Base is a near-white with a whisper of the palette's own warmth or coolness. Never pure white, never a shared flat gray.

Contrast is a rule, not a hope

The default button is Ink with Base text — always passes, always looks expensive. Colored buttons only when the Lead passes 4.5:1; otherwise use the palette's darkened variant.

Lead lives in the headline

Rather than a colored button, color one word of the headline in Lead. Editorial, expensive, zero contrast risk. The display-only Leads (Rose, Teal, Marigold) exist for exactly this.

Referent-first names

Palettes are named for real things with inherent color stories — a drink, a place, a plant — not adjectives. The referent hands the Spice over for free and gives clients a name to keep.

Ink always has a hue

Never pure black. Forest, plum, petrol, olive espresso — the hue keeps dark sections feeling like the brand instead of a void.

Do

Pair a Lead-colored headline word with Ink body on Base.

Run testimonial strips and feature bands on Paper.

Highlight key phrases with the Wash, Ink text on top.

Make primary buttons Ink; use Lead buttons only where the pairing table allows.

Use the listed darkened variants for any small Lead or Spice text.

Don't

Set Wash, Paper, or any display-only accent as text on a light background.

Introduce hues from outside the palette, including retired decoratives.

Use more than three text colors in one layout.

Put dark text on a mid-saturation colored button.

Collapse Base and Paper into one surface — they do different jobs.

Type pairings

Color and typography are two independent dials. Each palette above was shown in its quadrant's default pairing, but any of the four pairings works with any of the thirteen palettes. Pick the one that sounds like the brand's voice, not the color it happens to sit next to on the map. Each pairing also carries a matched accent script, chosen to belong to that pairing's register — an optional signature layer for eyebrows, category labels, and closings, always used sparingly.

The four defaults below are each a display serif for headlines over a sans-serif workhorse for body, and every serif display face has a true italic that carries the accent voice. Because every default puts a serif in the display slot, the whole set leans one way — so a second set of sans-serif display pairings follows, for brands that want a more modern or graphic voice up top. All faces are Google Fonts. Pairings are proposals pending Whitney's confirmation.

Use them sparingly — one word or one short line, at display sizes only, kept out of body copy and buttons. The defaults above are starting points, not the only right answers.

Sans-serif display options

All four defaults put a serif in the display slot, so the set reads one way. These four keep the same quadrant voices but swap the headline to a sans-serif display face — a more graphic, contemporary register for brands the serifs feel too traditional for. The body stays exactly as it was: each pairing reuses its quadrant's proven, WCAG-verified body sans (Karla, Plus Jakarta Sans, Work Sans, Archivo), so only the display voice changes. Because these display faces have no true italic, the accent voice is carried by weight and color instead of a slanted cut — a deliberate trade, and part of why they read more modern. Each is shown at heading, subheading, and body sizes.
